# MotoShop - Интернет-магазин мотоциклов и запчастей

## Описание проекта
MotoShop - это интернет-магазин мотоциклов и запчастей с возможностью просмотра каталога, добавления товаров в корзину и оформления заказов.

## Технологии
- Backend: NestJS
- Frontend: HTML, CSS, JavaScript (чистый)
- Шаблонизатор: EJS
- Хостинг: Render
